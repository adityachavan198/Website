Autonomous Home Security:


An Autonomous Home-Security system made using raspberry pi – 3 which detects motion outside the house and on motion detection clicks the photo of the person in the frame and sends the photo from the raspberry to the server (made using Django) where the processing(face recognition) happens. The server checks whether the person is there in the database of authorized users and if true then server return the result and the user is granted access. During the entire process the person standing outside the house can see the status of what is happening by LEDs as well as voice. Also the owner of the house receives notification on his android app as well as on his/her email..
